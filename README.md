# LMS

# Project dependencies
1. docker
2. docker-compose
3. poetry
4. make

# Setup Project
1. `git clone git@github.com:lukecyx/lms.git`
2. `cd lms`
3. `make build`
4. `make setup`

# Run project
1. `make runserver`
2. Go to `http://lms.local` in your web browser.

# Commands
1. Enter `make help` in your terminal.
